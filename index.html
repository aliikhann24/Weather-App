About Skies Weather

Skies is a modern, real-time weather application built as part of Muhammad Ali's frontend developer portfolio. It demonstrates the ability to work with live REST APIs, build responsive UIs, handle asynchronous JavaScript, and craft polished user experiences — all skills essential for a junior frontend developer role. This project is built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no libraries.

🌐
Live API Integration
Fetches real-time weather data from OpenWeatherMap's REST API using the Fetch API and modern async/await patterns. Handles loading states, API errors, and edge cases gracefully throughout.
📍
Geolocation Support
Uses the browser's native Geolocation API to detect your exact position and automatically load local weather — no manual input needed. Falls back gracefully if permission is denied.
🌡️
°C / °F Unit Toggle
Dynamic temperature unit switching that recalculates all values instantly client-side — no extra API calls needed. Demonstrates clean state management in plain JavaScript.
📅
5-Day Forecast + Hourly
Parses and groups OpenWeatherMap's 3-hour forecast data into clean daily summaries and hourly breakdowns using custom data aggregation logic written from scratch.
📱
Fully Responsive
Mobile-first layout using CSS Grid and Flexbox ensures a seamless, polished experience across all screen sizes — from small phones to large desktop monitors.
✨
Premium CSS Animations
Animated background blobs, floating icons, smooth page transitions, and staggered reveals all built with pure CSS keyframes — no external animation libraries.
🌟 Weather Knowledge Guide
01

Understanding Feels Like Temperature

The "feels like" value accounts for both wind chill and humidity. On humid days, your body cannot cool itself efficiently through sweat evaporation, making it feel hotter than the actual reading. On windy days, heat is stripped from your skin faster, making it feel colder. This value is often more useful than raw temperature for deciding what to wear.

02

Reading Atmospheric Pressure

Standard sea-level pressure is around 1013 hPa. A falling pressure reading typically signals incoming rain or storms on the way, while rising pressure means the weather is improving. A sudden drop of 10+ hPa within 3 hours often precedes severe weather events — worth monitoring if you're planning outdoor activities.

03

Humidity & Human Comfort

Humidity between 40–60% is the sweet spot for human comfort. Above 70%, the air feels sticky and heavy as sweat evaporates too slowly to cool you. Below 30%, the air becomes dry and can irritate your skin and airways. In cities like Karachi, summer humidity regularly exceeds 80%, making a 35°C day feel well above 42°C in practice.

04

Wind Speed & Safety Levels

Winds up to 20 km/h are pleasant and barely noticeable during outdoor activities. At 40–60 km/h, walking into the wind becomes difficult and loose objects can become hazardous. Above 90 km/h enters hurricane-force territory. Wind direction also matters — a sea breeze brings cooling moisture, while a land breeze brings dry, hot air inland toward the coast.

05

Visibility & Driving Safety

Visibility above 10 km is considered excellent and poses no issues. Between 1–5 km indicates fog or haze, where driving requires extra caution and reduced speeds. Below 200 meters is classified as dense fog — avoiding driving entirely is the safest choice. Visibility drops are most common at dawn, after heavy rain, and in coastal areas during winter months.
