About Skies Weather

Skies is a modern, real-time weather application built as part of Muhammad Ali's frontend developer portfolio. It demonstrates the ability to work with live REST APIs, build responsive UIs, handle asynchronous JavaScript, and craft polished user experiences — all skills essential for a junior frontend developer role. Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no libraries.

🌐
Live API Integration
Fetches real-time weather from OpenWeatherMap's REST API using Fetch API and async/await. Handles loading states and errors gracefully.
📍
Geolocation Support
Uses the browser's Geolocation API to detect your position and load local weather automatically. Falls back gracefully if denied.
🌡️
°C / °F Toggle
Temperature unit switching that recalculates all values instantly client-side — clean state management in plain JavaScript.
📅
5-Day Forecast + Hourly
Parses OpenWeatherMap 3-hour data into daily summaries and hourly breakdowns using custom aggregation logic.
📱
Fully Responsive
Mobile-first CSS Grid and Flexbox layout works seamlessly from small phones to large desktop monitors.
✨
CSS Animations
Animated blobs, floating icons, smooth transitions — all built with pure CSS keyframes, zero libraries.
🌟 Weather Knowledge Guide
01

Understanding Feels Like Temperature

The "feels like" value accounts for wind chill and humidity. On humid days your body can't cool efficiently, making it feel hotter. On windy days, heat leaves faster, making it feel colder than the actual reading.

02

Reading Atmospheric Pressure

Normal sea-level pressure is ~1013 hPa. Falling pressure means rain or storms incoming. Rising pressure means improving weather. A drop of 10+ hPa in 3 hours often precedes severe weather.

03

Humidity & Human Comfort

40–60% humidity is ideal. Above 70% feels muggy and heavy. In Karachi, summer humidity regularly exceeds 80%, making 35°C feel like 42°C or more.

04

Wind Speed & Safety

Up to 20 km/h is pleasant. At 40–60 km/h walking becomes difficult. Above 90 km/h is hurricane territory. Sea breezes bring cool moisture; land breezes bring dry heat.

05

Visibility & Driving Safety

Above 10 km is excellent. Between 1–5 km is fog or haze — drive carefully. Below 200 meters is dense fog — avoid driving. Most common at dawn and after rain.
